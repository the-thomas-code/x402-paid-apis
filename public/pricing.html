💰 Payment Method

Payments are processed via the x402 Protocol - HTTP-native crypto payments with instant settlement.

🔗 USDC on Base ⚡ 2-second settlement 🔐 No accounts needed 🤖 AI-agent ready

How It Works:

  1. Send a request to the API endpoint
  2. Receive HTTP 402 with payment instructions
  3. Sign payment with your wallet
  4. Retry request with payment signature
  5. Get instant access to the data
// Example with x402 client import { withPaymentInterceptor } from '@x402/core/client'; const api = withPaymentInterceptor(fetchClient, { privateKey: 'YOUR_KEY' }); const balance = await api.get('/api/balance?address=0x...');
